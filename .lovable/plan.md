

## תוכנית: מערכת הקלטות קוליות מותאמות אישית

### מה ייבנה
מערכת הקלטה קולית שמאפשרת להקליט הודעות קוליות (כמו "כל הכבוד!", "יופי!") ולשייך אותן לאירועי משחק (התאמה מוצלחת, ניצחון, היפוך קלף). ההקלטות נשמרות בענן ונגישות בין הפעלות.

### שלבים

**1. טבלת מסד נתונים חדשה — `voice_recordings`**
- עמודות: `id`, `device_id`, `name` (שם ההקלטה), `event_type` (match / win / flip / mismatch), `audio_url` (קישור לקובץ בענן), `is_active` (האם פעיל), `created_at`
- RLS פתוח (כמו שאר הטבלאות בפרויקט)

**2. רכיב ניהול הקלטות — `VoiceRecorder.tsx`**
- ממשק הקלטה עם כפתור הקלטה/עצירה (שימוש ב-MediaRecorder API)
- אינדיקציה חזותית בזמן הקלטה (אנימציית פעימה)
- רשימת הקלטות קיימות עם: השמעה, מחיקה, בחירת אירוע (match/win/flip/mismatch)
- העלאת הקלטה ל-bucket `game-audio` בענן
- toggle פעיל/לא פעיל לכל הקלטה

**3. שילוב בטאב הגדרות — `CardSetSelect.tsx`**
- הוספת טאב חדש "הקלטות" (🎙️) או שילוב בטאב המוזיקה הקיים
- תצוגת כל ההקלטות מקובצות לפי סוג אירוע

**4. שילוב במשחק — `useMemoryGame.ts` + `GameBoard.tsx`**
- טעינת הקלטות פעילות מהענן בתחילת משחק
- באירוע match/win/mismatch — אם יש הקלטה פעילה, להשמיע אותה במקום או בנוסף לצליל המובנה
- שימוש ב-`Audio()` API להשמעת קבצי האודיו מהענן

### פרטים טכניים
- הקלטה ב-WebM/OGG via `MediaRecorder`
- אחסון ב-bucket `game-audio` (כבר קיים ופומבי)
- `GameSettings` יקבל שדה `customVoiceEnabled: boolean`
- עמודת `custom_voice_enabled` בטבלת `game_settings`
- הפונקציה `playCustomVoice(eventType)` תטען ותשמיע הקלטה מתאימה
